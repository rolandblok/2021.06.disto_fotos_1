## toeltjes om lasers te calibreren

In process kan je per laser een foto invoeren.

## 2 tools
- batch.py : alle fotos in process.py worden gemodeleerd, data gaat naar disto_result.csv
    - let wel op dat je die naam moet aanpassen in script

- process.py : zoek spots, modeleer,
    - met pijltje naar andere files.
    - alle files invoeren in laser_photos.append


##Notes : 
- met (l) zoekt ie zelf de spots
- vergeet niet te sorteren als je de laser punten hebt (s)
- je kan de de spots saven met hoofdletter S (S), voor het modeleren.
